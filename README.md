# MoneyMe

Персональний веб-додаток обліку фінансів для ПК: рахунки (готівка/карти), витрати й надходження за статтями,
перекази між рахунками з комісією (% або фікс) та ручним курсом, аналітика, бекап у JSON.

## Стек

- **Backend**: Node.js + Express + better-sqlite3 (TypeScript)
- **Frontend**: React + Vite + TypeScript + Tailwind CSS
- Drag-and-drop переказів — `@dnd-kit`, графіки — `recharts`, запити — `@tanstack/react-query`

## Запуск (потрібен Node.js 18+ та інтернет для першого `npm install`)

```bash
cd MoneyMe
npm install        # ставить залежності для server і web (потрібен інтернет один раз)
npm run dev        # піднімає API (http://localhost:4000) і фронт (http://localhost:5173)
```

Відкрийте http://localhost:5173

Дані зберігаються локально у `server/data/moneyme.db`. Бекап — кнопки «Експорт/Імпорт JSON» у Налаштуваннях.

## Скрипти

- `npm run dev` — сервер + фронтенд разом
- `npm run dev:server` / `npm run dev:web` — окремо
- `npm run typecheck` — перевірка типів
- `npm test` — юніт-тести розрахунків (`server`)

## Ключові можливості

- Кілька рахунків з різними валютами та **стартовими залишками**.
- Швидке додавання: кнопки **«+»** (надходження) і **«−»** (витрата).
- Переказ **перетягуванням** однієї картки рахунку на іншу.
- Комісія за переказ (% або фіксована) ведеться **окремим підсумком**, не змішується з витратами.
- Аналітика за статтями + зведення за період.
